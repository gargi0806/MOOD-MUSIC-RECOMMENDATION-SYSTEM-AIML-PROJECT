# 🎵 Mood Music Recommendation System

A music recommendation system that suggests songs based on the user's mood using Machine Learning and Python.

## 📌 About the Project
This project detects the user's current mood as input and recommends suitable songs accordingly. It uses machine learning techniques to match emotions with music genres and tracks.

## ✨ Features
- 🎭 Mood-based song recommendations
- 🎨 Interactive user interface
- 🎼 Music filtering by emotion/genre
- 🤖 ML-powered mood detection

## 🛠️ Tech Stack
- Python
- Machine Learning
- Pandas
- Scikit-learn
- Flask

## 🚀 How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
